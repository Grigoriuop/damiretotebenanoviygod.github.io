⚡ Pehkui Команды

Полный справочник всех параметров изменения размера | Minecraft Mod

📋 Основные команды

🎯 Установить масштаб

Устанавливает значение для указанного типа масштаба

/scale set pehkui:base 0.5 @p

➕ Прибавить к масштабу

Увеличивает текущий масштаб на указанное значение

/scale add pehkui:base 0.3 @p

➖ Вычесть из масштаба

Уменьшает текущий масштаб на указанное значение

/scale subtract pehkui:base 0.2 @p

✖️ Умножить масштаб

Умножает текущий масштаб на указанное значение

/scale multiply pehkui:base 1.5 @p

➗ Разделить масштаб

Делит текущий масштаб на указанное значение

/scale divide pehkui:base 2 @p

🔍 Получить текущий масштаб

Показывает текущее значение масштаба

/scale get pehkui:base @p

🧮 Вычислить итоговый масштаб

Показывает итоговый масштаб с учётом всех модификаторов

/scale compute pehkui:base @p

🔄 Сбросить масштаб

Сбрасывает масштаб к значению по умолчанию (1.0)

/scale reset @p

⏱️ Установить задержку изменения

Время в тиках (20 тиков = 1 секунда) на плавное изменение размера

/scale delay set pehkui:base 40 @p

💾 Сохранять размер после смерти

true — размер сохранится после возрождения, false — сбросится

/scale persist set pehkui:base true @p

📐 Размер и модель

📏 Базовый размер

Основной масштаб, влияет на все остальные параметры

/scale set pehkui:base 1.5 @p

📐 Ширина модели

Изменяет ширину/длину/глубину модели сущности

/scale set pehkui:model_width 0.7 @p

📏 Высота модели

Изменяет высоту модели сущности

/scale set pehkui:model_height 0.7 @p

📦 Ширина хитбокса

Изменяет ширину/длину/глубину хитбокса

/scale set pehkui:hitbox_width 0.6 @p

📦 Высота хитбокса

Изменяет высоту хитбокса сущности

/scale set pehkui:hitbox_height 0.6 @p

👁️ Высота глаз

Изменяет высоту положения глаз сущности (точка обзора)

/scale set pehkui:eye_height 0.8 @p

🎥 Дистанция камеры от 3-го лица

Как далеко камера от игрока в режиме 3-го лица

/scale set pehkui:third_person 1.3 @p

🖐️ Масштаб предмета в руке

Размер предметов, которые держит игрок

/scale set pehkui:held_item 1.2 @p

🏃 Движение и физика

💨 Скорость движения

Изменяет скорость передвижения (ходьба, бег)

/scale set pehkui:motion 1.5 @p

🦶 Высота шага

Позволяет забираться на блоки выше без прыжка

/scale set pehkui:step_height 1.5 @p

🕹️ Покачивание вида

Влияет на эффект покачивания камеры при ходьбе

/scale set pehkui:view_bobbing 1.2 @p

💔 Урон от падения

Изменяет получаемый урон при падении

/scale set pehkui:falling 0.5 @p

🕊️ Скорость полёта (креатив)

Изменяет скорость полёта в творческом режиме

/scale set pehkui:flight 2.0 @p

🦘 Высота прыжка

Изменяет высоту прыжка

/scale set pehkui:jump_height 1.3 @p

🤝 Досягаемость и взаимодействие

🎯 Общая досягаемость

Влияет на дистанцию взаимодействия с блоками и сущностями

/scale set pehkui:reach 2.0 @p

🧱 Досягаемость до блоков

Дистанция, с которой можно ломать/ставить блоки

/scale set pehkui:block_reach 1.8 @p

⚔️ Досягаемость атаки

Дистанция, с которой можно ударить моба/игрока

/scale set pehkui:entity_reach 1.6 @p

👁️ Видимость для врагов

Дистанция, с которой мобы замечают игрока

/scale set pehkui:visibility 0.5 @p

⚔️ Бой и характеристики

💥 Наносимый урон

Изменяет урон, который наносит сущность

/scale set pehkui:attack 2.0 @p

🛡️ Получаемый урон

Изменяет урон, который получает сущность (защита)

/scale set pehkui:defense 0.5 @p

💨 Отбрасывание (нокбэк)

Сила отбрасывания при ударе

/scale set pehkui:knockback 1.5 @p

❤️ Здоровье

Изменяет максимальное количество сердец

/scale set pehkui:health 2.0 @p

⛏️ Скорость копания

Изменяет скорость разрушения блоков

/scale set pehkui:mining_speed 2.0 @p

🗡️ Скорость атаки

Изменяет перезарядку удара (как у меча)

/scale set pehkui:attack_speed 1.8 @p

💎 Предметы и спецэффекты

📦 Размер выпавших предметов

Размер предметов, лежащих на земле

/scale set pehkui:drops 0.8 @p

🏹 Размер снарядов

Размер стрел, снежков, яиц и других снарядов

/scale set pehkui:projectiles 1.3 @p

💥 Размер взрыва

Радиус взрыва крипера, кристалла Энда и т.д.

/scale set pehkui:explosions 2.0 @p

🛠️ Продвинутые операции

🎲 Случайный масштаб

Устанавливает случайное значение в указанном диапазоне

/scale randomize pehkui:base 0.5 2.0 @p

📋 Получить NBT масштаба

Показывает NBT-данные масштаба сущности

/scale nbt get @p

🔁 Сброс конкретного типа

Сбрасывает только указанный параметр

/scale reset pehkui:attack @p

⏱️ Получить задержку изменения

Показывает текущую задержку изменения масштаба

/scale delay get pehkui:base @p
📌 Как использовать: • @p — ближайший игрок | @s — вы | @a — все игроки | @e[type=zombie] — все зомби • Значение 1.0 — стандартный размер | <1 — меньше | >1 — больше • Для плавного изменения используйте задержку: /scale delay set pehkui:base 20 @p • Список всех параметров: base, width, height, hitbox_width, hitbox_height, eye_height, model_width, model_height, third_person, motion, step_height, view_bobbing, falling, flight, jump_height, visibility, reach, block_reach, entity_reach, held_item, drops, projectiles, explosions, attack, defense, knockback, health, mining_speed, attack_speed
